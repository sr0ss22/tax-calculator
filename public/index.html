Quote Tax Estimate

Estimate sales tax for a quote. Enter lines, a location, and optional fees. State takes a US state code or name, or a Canadian province name or code (for example Ontario or BC) to use the static Canada rates. This is an estimate; SAP remains the system of record for actual tax.

Location and channel

US rates use the built-in state average combined rate by default (no API). Enter a rate override for the exact local rate, or configure a TaxJar token for live ZIP lookup. Canada needs no rate (static chart).

Lines

Each line is a Product or an Install. Installation labor is taxed by per-category rules that differ by state, so add Install Drapery, Install Blinds, and Install Shutters as their own lines.
