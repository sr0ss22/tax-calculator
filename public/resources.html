Official State Rate Lookups

Each link opens that state's official rate-lookup tool. Enter the install address to get the exact combined rate, then key it into the quote as the rate override. Source: THD Window Tax Chart.

Alaska has no statewide tax — rate is by locality (Anchorage, Juneau, Kenai, Wasilla). Georgia rate varies by county. Delaware, Montana, New Hampshire, Oregon have no sales tax.

THD Window Tax ChartEffective 2025-03-21

Home Depot taxability matrix used for RLN quotes through the THD channel.

The chart flags taxability (yes/no), not rates — the rate comes from the state lookup above. THD's model taxes the installed package (product + install labor) as a unit, so within a product category the flags almost always move together (all taxable or all exempt).

  • Blinds & Draperies are taxable in most tax states; Shutters are frequently exempt (treated as a real-property improvement) — that mismatch is what triggers blended tax.
  • Iowa is the systematic exception: labor taxable, product not.
  • District of Columbia: Blinds & Draperies taxable; Shutters not applicable.
  • No-tax states: Delaware, Montana, New Hampshire, Oregon (plus Alaska = local only).
This is a maintained artifact (last reissued 2025-03-21). Always confirm against the current chart in Salesforce → Designer Links → Tax Rates - THD Sales before relying on it.

Channel Tax ChartUpdated 2024-03

Decorview / Macy's / DirectBuy / JCPenney taxability matrix.

The non-THD channels use a different model: they distinguish product-taxable from install-labor-taxable per state (commonly product Yes / install No). Install labor is taxable in only ~17 states here, versus ~30 jurisdictions for THD. Several states flip between the two charts — e.g. California is fully non-taxable on THD but product-taxable on the channel chart; New Jersey is the reverse.

Pick the channel (THD vs Other) correctly in the calculator — the taxability flags differ by chart.

Blended Tax Guide

A THD-only order-entry workaround — not a state-law concept.

When it applies: a single THD job mixes Blinds and Shutters in a state where their taxability differs (e.g. Texas — blinds taxable, shutters exempt). Because order entry takes one rate for the whole order, you enter a single weighted-average rate.

How the rate is figured:

  1. Tax each line by its own taxability (taxable lines × the local rate; exempt lines = $0 tax).
  2. Blended rate = total tax ÷ total retail across the whole order.
  3. Enter that one rate on both the TAX and TAX-Install lines.
  4. Add the mandatory Salesforce Note titled "THD Blinds and Shutters" recording the inputs.
The calculator computes this automatically — when a blended case is detected it shows the Blended tax rate to key in. This page is the reference for what it's doing.

Download Rates & Enter on a Quote

The current Salesforce workflow for finding a rate and applying it to line items.

  1. On the Opportunity, copy the consumer's install address.
  2. Home tab → Designer Links → Tax Rates - THD Sales → Download.
  3. Open The Home Depot Window Tax Chart US PDF; find the state and confirm whether it's taxable by product category.
  4. Click the state's link (or use the State Rate Lookups above), paste the address, and read the combined rate.
  5. Back in the quote → Manage Line Items → enter the rate in Tax Rate % → Save and Close.
  6. Apply the same rate to any added installation/service lines.
